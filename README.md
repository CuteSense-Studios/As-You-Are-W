# As You Are – Official Website

Welcome to the repository for the official website of **"As You Are"**, a visual novel developed by **CuteSense Studios**.

This website serves as the landing page for the game, providing information about the story, characters, and development updates.

## 🚀 Project Overview

  - **Studio:** [CuteSense Studios](https://www.google.com/search?q=https://github.com/CuteSense-Studios)
  - **Author:** Crossie
  - **Technologies:** HTML5, CSS3 (including custom cursors and responsive design)
  - **License:** CC BY-SA 4.0

## 📂 Repository Structure

  - `index.html`: The main entry point of the website.
  - `style.css` / `input.css`: Core stylesheets for layout and design.
  - `cursor.css`: Custom cursor implementation for a unique user experience.
  - `assets/`: Directory for fonts and other web resources.
  - `images/`: Directory for game screenshots, character art, and branding.

## 🎨 Branding & Design

The website reflects the aesthetic of the "As You Are" visual novel.

  - **Logo Font:** Borel
  - **Theme:** Inclusive and narrative-driven, mirroring the game's focus on identity and self-discovery.
